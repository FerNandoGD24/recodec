# recodec
Basicamente recodifica los videos a un formato 100% compatible con davinci resolve en linux, el stereo se utiliza como mono, es tu trabajo modificar el audio para que use un solo canal, 1 pista en stereo se convierte en 2 pistas en mono, tu debes identificar cual es la pista izquerda y cual es la derecha y en davinci resolve lo conviertes a stereo
